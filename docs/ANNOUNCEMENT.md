# 公告：红色警戒2 增强 Mod 开源了

抖音上很多老玩家在找：**红警2 电脑更聪明一点、体验更顺一点**。  
我们正在做的，就是这件事——一个面向 **红色警戒2 / 尤里的复仇** 的开源增强 Mod：

**仓库（请分享这个）**：<https://github.com/Renhuai123/ra2-enhanced>

## 能做什么 / 不能做什么（请先看清）

红色警戒的知识产权属于 **EA**。  
因此我们**不能**、也**不会**去做一个「独立商业红警2 产品」去卖游戏本体或分发版权资源。

我们**可以**做的是：

- 以 **Mod（规则 / AI / QoL）** 的方式，优化老玩家熟悉的 **红警2** 体验
- 加强 **电脑 AI**（扩张、防守、兵种构成、持续压力）
- 后续继续探索 **更强的 AI 对手**——**敬请期待**

本仓库只包含 Mod 规则、脚本与文档，**不包含**游戏二进制或 `.mix` 等版权资源。  
游玩前请自行准备合法的 **红警2 + 尤里的复仇**，并按文档安装 Ares / Phobos 等社区扩展。

## 不是 OpenRA 红警1

此前有一个基于 OpenRA 官方 `ra`（红警1）的实验仓库，**方向错了**。  
红警2 ≠ OpenRA 里的 Red Alert 1。请以本仓库为准：

| 错误方向 | 正确方向 |
|----------|----------|
| OpenRA stock `ra`（红警1） | 原版红警2 / 尤里 + Ares + Phobos |

## 怎么玩（最短路径）

1. 自备合法 **红警2 + 尤里的复仇**
2. 安装 **Ares + Phobos**（官方/上游发布页）
3. `git clone` 本仓库，运行 `scripts/install-mod.ps1` 指向你的游戏目录
4. 用 Syringe / CnCNet 启动，进入遭遇战验证

详细步骤见根目录 [README.md](../README.md) 与 [install.md](install.md)。

我们不是要取代你记忆里的红警2，而是希望在合法、开源的前提下，让「再开一局」更过瘾：电脑不那么摆烂，界面与操作反馈更清楚。

欢迎 Star、提 Issue、一起改 AI——**敬请期待**后续更强的电脑与体验更新。
