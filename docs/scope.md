# Scope & legal boundary

## In scope

- Open-source INI overlays for RA2/YR (AI, QoL, documentation)
- Install helpers that copy/link **only** this repo's text assets
- Chinese / English docs for players and contributors

## Out of scope

- Distributing RA2/YR binaries or `.mix` / art / audio / video
- Selling a standalone commercial “红警2” client
- Claiming EA endorsement
- Pretending OpenRA `ra` (RA1) is Red Alert 2

## Dependencies (user-supplied)

| Component | Source |
|-----------|--------|
| Game | Player-owned RA2 + YR |
| Ares + Syringe | launchpad.net/ares |
| Phobos | github.com/Phobos-developers/Phobos |
| CnCNet client (optional) | cncnet.org / upstream packages |
